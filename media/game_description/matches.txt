Matches is a simple game in which you should smartly select number of matches. Choose your own strategy for win.

What the rules?
You have 21 match on the playing field. You and your opponent take turns choosing the number of matches from one to three. Whoever takes the last match loses.

Below you can see .txt file with template of bot. Copy it in your .py file and begin to create your own winner. Remind: bot must be written on Python!
